About This Site

This is a portfolio website that showcases my work as a software engineer. It is inspired by neural networks, allowing you to explore my projects as if they were nodes. The "About me" page features 3D models that represent me and my interests, displaying statistics about me in the same way as video-game statistics. This was inspired by most of my projects surrounding gaming and game development.

Technologies Used

  • HTML5 + Bootstrap
    This was used for the structure of the website
  • CSS + Bootstrap
    This was used for the styling of the website
  • Blender
    This was used to create the meshes, textures and animations for the models
  • JavaScript + Three.js
    This was used for the 3D aspects of the website,
